Redis Cluster with Spring Data Redis
=========================

Building the Application
---------------------
    $ git clone https://github.com/buraksarp/redis-cluster-spring-data.git
    $ cd redis-cluster-spring-data
    $ mvn clean install

Setting Up the Redis Cluster
---------------------
